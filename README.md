# 🚀 Mi Portafolio Personal - UI/UX & Frontend

¡Bienvenido a mi portafolio! Esta es una aplicación web moderna construida con **Angular 21** y **Tailwind CSS**, diseñada bajo una filosofía de minimalismo, rendimiento y accesibilidad.

---

## ✨ Características Principales

- **Arquitectura Standalone:** Componentes independientes y ligeros sin necesidad de módulos pesados.
- **Modo Oscuro (Dark Mode):** Implementación con persistencia en `localStorage` y Angular Signals.
- **Scroll Reveal API:** Animaciones nativas de entrada mediante `Intersection Observer` para una experiencia fluida.
- **Diseño Responsivo:** Adaptado para móviles, tablets y escritorio siguiendo la metodología *Mobile-First*.
- **Formularios Reactivos:** Validación en tiempo real y estados de carga (Loading Spinners) para una mejor UX.

---

## 🛠️ Stack Tecnológico

| Tecnología | Uso |
| :--- | :--- |
| **Angular 21** | Framework principal (Standalone Components) |
| **TypeScript** | Programación robusta con Interfaces de datos |
| **Tailwind CSS** | Estilizado rápido y diseño responsivo |
| **Angular Signals** | Gestión de estado reactiva para el Modo Oscuro |
| **Vercel / Firebase** | Despliegue y hosting |

## 🚀 Instalación y Uso Local

Si deseas clonar este proyecto y ejecutarlo en tu máquina:

1. **Clona el repositorio:**
   ```bash
   git clone [https://github.com/tu-usuario/mi-portafolio.git](https://github.com/tu-usuario/mi-portafolio.git)

2. **Instala las dependencias:**

Bash
npm install

3. **Inicia el servidor de desarrollo:**

Bash
ng serve
Ve a http://localhost:4200 en tu navegador.

Microinteracciones: Efectos de hover y transiciones suaves (0.3s - 0.5s) para dar feedback constante.

Contraste: Cumplimiento con las normas WCAG de accesibilidad tanto en modo claro como oscuro.
